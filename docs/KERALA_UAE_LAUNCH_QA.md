# Kerala UAE Launch QA

## Buyer Flow

- Search Kerala properties.
- Search UAE properties.
- Save a property.
- Submit inquiry.
- Use AI search in English and Malayalam.

## Seller Flow

- Create property listing.
- Upload property media.
- Confirm listing enters review.
- Confirm dashboard shows listing status.

## Broker Flow

- Broker role dashboard loads.
- Broker listings show lead and performance metrics.
- Broker verification status is visible.
- Follow/unfollow works from creator profile.

## Builder Flow

- Builder dashboard loads.
- Projects and inventory are visible.
- Builder lead and media metrics update.
- Builder verification status is visible.

## Payment Flow

- Featured listing checkout starts.
- Premium listing checkout starts.
- Success requires Razorpay signature verification.
- Payment history appears in dashboard.

## AI Flow

- AI search returns DB properties only.
- Recommendations use saved/inquiry/view/search signals.
- Area recommendations disclose insufficient data.
- Lead assistant drafts only and does not send messages.
